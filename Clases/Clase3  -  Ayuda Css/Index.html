Niveles de jerarquia

De los 3 metodos (Archivo externo, etiqueta style y atributo style) El que mas jerarquia posee es el atributo style , luego la Etiqueta style y por ultimo el archivo externo.

Atributo class

se utiliza para darle estilos a ciertas partes mas especificas del codigo.
Desde css se hace mencion con el nombre del class que lleve la etiqueta a dar estilo en el HTML, y por delante de su nombre lleva un punto, lo recomnedable es colocar un nombre que haga referencia a los estilos que tendra. Ejemplo:
HTML :
css : .bordesredondeados {
}
Al agregar dos valores de class separados por un espacio, me permite ser mas especifico con mi codigo para modificarlo mas adelante.

Atributo ID

Generalmente se usa para nombrar porciones de codigo sectores, Es posible ponerle ID a cualquier elemento HTML para darle "nombre". y asi como el ID, todos los elementos tambien aceptan el atributo class="".
Dicha clase se utiliza cuando quieres aplicar el mismo estilo a mas de un elemnto, y la busqueda por etiqueta no sirve lograrlo.

ID desde css

Desde CSS, se puede utilizar cualquier nombre que empiecen con letras, y delante se coloque un numeral (#). Lo recomendable es poner un nombre que haga referencias a los estilos que tendra. por ejemplo:
#productos {
}
Para hacer mencion en HTML se coloca el atributo ID="" en la etiqueta deseada
